# Mytory Video Tool (데스크톱 통합 앱)

**Mytory Video Tool**은 브라우저 WebAssembly의 2GB 메모리 제한을 완벽히 극복하고, 네이티브 FFmpeg을 활용하여 대용량 비디오의 배속 인코딩 및 다양한 미디어 조작을 지원하는 크로스 플랫폼(Windows, macOS, Linux) 데스크톱 애플리케이션입니다.

---

## 1. 주요 특징 및 제공 기능

1. **WASM 한계 극복**: 로컬 시스템의 CPU/GPU 자원을 그대로 활용하여 메모리 크래시 없이 테라바이트급 고해상도 영상을 안정적으로 처리합니다.
2. **사이드바 탭 네비게이션**: 5가지 핵심 미디어 유틸리티를 하나의 세련된 다크 테마 대시보드 안에서 자유롭게 전환하며 사용할 수 있습니다.
3. **유연한 저장 정책**: 기본적으로 원본 비디오 파일과 동일한 경로에 작업 접미사를 붙여 고속 저장하며, 사용자가 원하는 커스텀 출력 폴더를 별도로 구성할 수 있습니다.
4. **하드웨어 가속 자동 감지**: 앱이 시작할 때 로컬 GPU 사양을 스캔하여 플랫폼별 최적의 하드웨어 인코더(Apple Silicon VideoToolbox, NVIDIA NVENC, Intel QSV, AMD AMF 등)를 기본값으로 자동 셋팅합니다.

### 5가지 핵심 도구
* **⚡ 배속 변환기 (Speed Changer)**: 다람쥐 목소리 방지 기술(피치 유지)을 내장하여 0.5x ~ 4.0x 속도로 비디오 배속 인코딩을 수행합니다. (H.264, H.265/HEVC, VP9, AV1 코덱 지원)
* **🎵 오디오 추출기 (Audio Drop)**: 비디오 파일에서 오디오 트랙만 무손실 복사(Auto)하거나 다양한 형식(MP3, AAC, OGG, WAV)으로 변환해 추출합니다.
* **📸 장면 캡처 (Frame Capture)**: 프레임 단위 탐색 캡처, 지정 간격(Batch) 연속 추출 및 민감도 조절을 통한 장면 전환(Scene Detection) 자동 추출 기능을 제공합니다.
* **🔄 확장자 변환기 (Remuxer)**: 재인코딩 과정 없이 고속으로 비디오의 컨테이너 포맷(MP4, MKV, MOV)만 변경합니다.
* **✂️ 비디오 분할기 (Video Splitter)**: 인코딩 손실 없이 비디오의 원하는 영역 시작점과 종료점을 선택해 매우 빠르게 자릅니다.

## 2. ⚠️ 설치 및 실행 시 주의사항 (보안 경고)

본 프로그램은 개인 개발자가 배포하는 오픈소스 소프트웨어로, 고가의 유료 개발자 인증서 서명이 되어 있지 않습니다. 이에 따라 설치 및 실행 시 다음과 같은 보안 경고가 발생할 수 있으나, 이는 프로그램의 결함이 아니니 안심하고 사용하셔도 됩니다.

### Windows
*   **증상:** 실행 시 "Windows의 PC 보호" 파란색 창이 뜹니다.
*   **해결 방법:** 창 내의 **[추가 정보]** 링크를 클릭한 후 나타나는 **[실행]** 버튼을 누르면 정상적으로 설치/실행됩니다.

### macOS
*   **증상:** "확인되지 않은 개발자" 또는 "악성 소프트웨어가 있는지 확인할 수 없음" 메시지가 뜹니다.
*   **해결 방법:**
    1.  `.dmg` 설치 후 앱 파일을 실행할 때 **마우스 오른쪽 버튼(또는 Control + 클릭)**을 누릅니다.
    2.  메뉴에서 **[열기]**를 선택합니다.
    3.  다시 나타나는 확인 창에서 **[열기]**를 한 번 더 누르면 이후부터는 정상 실행됩니다.

## 3. 시작하기

### 종속성 설치
로컬 환경에 [Node.js](https://nodejs.org/)가 설치되어 있어야 합니다.

```bash
# 의존성 패키지 설치
npm install
```

### 개발 모드로 실행
```bash
# 애플리케이션 실행
npm start
```

### 배포 패키지 빌드
`electron-builder`를 사용하여 배포용 단일 실행 파일(.dmg, .exe, .AppImage)을 빌드합니다.

```bash
# 디렉터리 구조로 빌드 테스트
npm run pack

# 최종 배포 파일 생성 (dist/ 폴더 내에 생성됨)
npm run dist
```

```text

    ├── app.css           # LCH 다크 모드 스타일
    ├── app.js            # UI 상태 및 드랙앤드롭 제어
```

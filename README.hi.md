# Mytory Video Tools

> **🇬🇧 English:** [README.md](./README.md) · **🇰🇷 한국어:** [README.ko.md](./README.ko.md) · **🇯🇵 日本語:** [README.ja.md](./README.ja.md) · **🇨🇳 简体中文:** [README.zh-cn.md](./README.zh-cn.md) · **🇪🇸 Español:** [README.es.md](./README.es.md) · **🇧🇷 Português:** [README.pt.md](./README.pt.md) · **🇫🇷 Français:** [README.fr.md](./README.fr.md) · **🇮🇩 Bahasa Indonesia:** [README.id.md](./README.id.md)

**Mytory Video Tools** एक क्रॉस-प्लेटफ़ॉर्म (Windows, macOS, Linux) डेस्कटॉप एप्लिकेशन है जो हाई-स्पीड वीडियो एन्कोडिंग और विभिन्न मीडिया संचालन कार्यों के लिए है। यह Electron के साथ बनाया गया है और इसमें एम्बेडेड FFmpeg/FFprobe बाइनरी शामिल हैं।

---

## 1. विशेषताएँ

1. **साइडबार टैब नेविगेशन**: एक आकर्षक डार्क-थीम डैशबोर्ड के भीतर 7 मुख्य मीडिया उपयोगिताओं के बीच स्विच करें।
2. **लचीली आउटपुट नीति**: डिफ़ॉल्ट रूप से, फ़ाइलें कार्य-विशिष्ट प्रत्यय के साथ मूल फ़ाइल के साथ सहेजी जाती हैं। कस्टम आउटपुट फ़ोल्डर भी कॉन्फ़िगर किए जा सकते हैं।
3. **स्वचालित हार्डवेयर त्वरण पहचान**: स्टार्टअप पर, ऐप आपके GPU को स्कैन करता है और आपके प्लेटफ़ॉर्म के लिए इष्टतम हार्डवेयर एन्कोडर का चयन करता है (Apple Silicon VideoToolbox, NVIDIA NVENC, Intel QSV, AMD AMF, आदि)।

### 7 मुख्य उपकरण

| उपकरण | विवरण |
|---|---|
| ⚡ **गति परिवर्तक** | पिच संरक्षण के साथ वीडियो प्लेबैक गति बदलें (0.5x ~ 4.0x)। H.264, H.265/HEVC, VP9, AV1 का समर्थन करता है। |
| 🎵 **ऑडियो निष्कर्षण** | ऑडियो ट्रैक को बिना गुणवत्ता हानि के निकालें या MP3, AAC, OGG, या WAV में बदलें। |
| 📸 **फ्रेम कैप्चर** | एकल फ्रेम कैप्चर, अंतराल पर बैच निष्कर्षण, और संवेदनशीलता नियंत्रण के साथ स्वचालित दृश्य पहचान। |
| 🔄 **रीमक्सर** | बिना पुनः एन्कोडिंग के तेज़ कंटेनर प्रारूप रूपांतरण (MP4, MKV, MOV)। |
| 🔗 **वीडियो जोड़ें** | समान एन्कोडिंग पैरामीटर वाली कई वीडियो फ़ाइलों को बिना गुणवत्ता हानि के जोड़ें। विभाजित खंडों को फिर से जोड़ने के लिए उपयुक्त। |
| ✂️ **वीडियो विभाजक** | प्रारंभ और समाप्ति बिंदु सेट करके बिना गुणवत्ता हानि के खंड काटें। अत्यंत तेज़। |
| 📦 **संपीड़क** | विभिन्न कोडेक और गुणवत्ता विकल्पों के साथ वीडियो को लक्ष्य फ़ाइल आकार में संपीड़ित करें। |

> **🖱️ वैश्विक ड्रैग और ड्रॉप**: फ़ाइलों को ऐप विंडो पर कहीं भी खींचें और वर्तमान में सक्रिय उपकरण में तुरंत जोड़ें।

## 2. 🚀 अंतिम उपयोगकर्ता: स्थापना और चलाना

यदि आप एप्लिकेशन को स्वयं बनाए बिना उपयोग करना चाहते हैं, तो नीचे दिए गए चरणों का पालन करें।

### डाउनलोड
*   [GitHub Releases](https://github.com/mytory/mytory-video-tools/releases) पेज पर जाएँ और अपने ऑपरेटिंग सिस्टम के लिए पैकेज डाउनलोड करें।
    *   **Windows:** `.exe` (इंस्टॉलर)
    *   **macOS:** `.dmg` (डिस्क इमेज) — Intel (x64) / Apple Silicon (arm64)
    *   **Linux:** `.AppImage` (पोर्टेबल)

### ⚠️ सुरक्षा चेतावनी (कृपया पढ़ें)
यह प्रोग्राम एक व्यक्तिगत डेवलपर द्वारा वितरित किया गया है और यह भुगतान कोड साइनिंग प्रमाणपत्र से हस्ताक्षरित नहीं है। आप जो सुरक्षा चेतावनी देख सकते हैं वह **कोई दोष नहीं है**। एप्लिकेशन चलाने के लिए नीचे दिए गए निर्देशों का पालन करें।

*   **Windows:** लाल SmartScreen चेतावनी पर, **[अधिक जानकारी]** → **[फिर भी चलाएँ]** पर क्लिक करें
*   **macOS:** ऐप फ़ाइल पर **राइट-क्लिक (या Control+क्लिक)** करें → **[खोलें]** चुनें → फिर से **[खोलें]** पर क्लिक करें

## 3. 🛠 डेवलपर्स: स्रोत से निर्माण

डेवलपमेंट एनवायरनमेंट सेट करने और प्रोजेक्ट बनाने के निर्देश।

### आवश्यक शर्तें
आपके सिस्टम पर [Node.js](https://nodejs.org/) स्थापित होना चाहिए।

### निर्भरताएँ स्थापित करें
```bash
npm install
```

### डेवलपमेंट मोड में चलाएँ
```bash
npm start
```

### वितरण के लिए बनाएँ
```bash
# dist/ फ़ोल्डर में वितरण योग्य पैकेज बनाता है
npm run dist
```

### macOS कोड साइनिंग / CI बिल्ड नोट्स

* `npm run dist` `dotenv` के माध्यम से `.env` से एनवायरनमेंट वेरिएबल लोड करता है।
* macOS साइनिंग आइडेंटिटी `.env` में `CSC_NAME` वेरिएबल से पढ़ी जाती है। यह वेरिएबल आपके स्थानीय macOS कीचेन में स्थापित प्रमाणपत्र का नाम निर्दिष्ट करता है।
* `.env` उदाहरण:
    ```env
    CSC_NAME="Apple Development: my-email@test.com (XXXXJ356NG)"
    ```
* `.github/workflows/release.yml` में `CSC_LINK` / `CSC_KEY_PASSWORD` GitHub Actions वातावरण में P12-आधारित हस्ताक्षर के लिए उपयोग किए जाते हैं।
